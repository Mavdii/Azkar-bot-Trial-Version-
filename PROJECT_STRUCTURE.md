## 📋 **وصف المجلدات**

### 🔧 **الملفات الأساسية**
- **premium_Azkar.py**: الملف الرئيسي الذي يحتوي على كامل منطق البوت
- **config.py**: إدارة الإعدادات والمتغيرات البيئية
- **islamic_bot_database.sql**: ملف SQL لإنشاء قاعدة البيانات الكاملة

### 🖼️ **مجلدات الصور**
- **images/**: صور أذكار الصباح والمساء الثابتة
- **post_prayer_images/**: صور أذكار ما بعد الصلاة
- **quran_pages/**: صفحات القرآن الكريم (604 صفحة)
- **random_images/**: صور متنوعة للأذكار العشوائية

### 📝 **مجلدات البيانات**
- **logs/**: ملفات السجلات والتتبع
- **backups/**: النسخ الاحتياطية لقاعدة البيانات
- **cache/**: ملفات التخزين المؤقت لتحسين الأداء
- **temp/**: ملفات مؤقتة أثناء المعالجة

### 🛠️ **مجلدات التطوير**
- **tests/**: اختبارات الوحدة والتكامل
- **docs/**: الوثائق والأدلة
- **scripts/**: سكريبتات المساعدة والأتمتة
- **monitoring/**: إعدادات المراقبة والتحليل

### 🚀 **مجلدات النشر**
- **.github/**: إعدادات GitHub Actions للـ CI/CD
- **monitoring/**: إعدادات Prometheus وGrafana

## 🔒 **ملفات .keep**

تم إضافة ملفات `.keep` في جميع المجلدات لضمان:
- **تتبع Git**: حفظ هيكل المجلدات في نظام التحكم بالإصدارات
- **النشر السليم**: ضمان وجود المجلدات عند النشر
- **التوثيق**: شرح الغرض من كل مجلد

- **📁 المجلدات**: 12 مجلد رئيسي

## 🎯 **الاستخدام**

### للمطورين:
```bash
# استنساخ المشروع
git clone https://github.com/yourusername/islamic-telegram-bot.git
cd islamic-telegram-bot

# إعداد البيئة
cp .env.example .env
pip install -r requirements.txt

# تشغيل البوت
python premium_Azkar.py
```

### للنشر:
```bash
# باستخدام Docker
docker-compose up -d

# أو النشر المباشر
python premium_Azkar.py
```
